Enterprise Strategic Planning

Designing new dashboard creation functionality

Interaction design

The project

Helping managers build charts and dashboards that provide timely strategic information to support decision-making at the C-level.

The problem

When I joined the project, the product's executive dashboard and data manipulation tools were only available on the iPad, and the single dashboard came with a limited set of pre-built charts. Usability testing had indicated that users would prefer to do the data manipulation on a laptop or desktop, and wanted the flexibility to build their own charts and dashboards.

The solution

Under the guidance of our UX lead, the visual designer and I created a desktop interface for users to build their own dashboards and viewlets (widgets) based on data stored in a multidimensional database. To keep the interface as clean as possible, we used a number of interaction patterns, including drag-and-drop capabilities, tabs, scrolling panels, filters, and hover panes.
